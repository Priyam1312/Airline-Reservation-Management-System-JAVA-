✈️ Airline Reservation and Management System (Java Console)

📌 Overview

This is a Java console-based Airline Reservation and Management System (ARMS) developed for academic purposes.
The system allows users to book flights, select seats, choose additional services, and manage basic airline operations using a menu-driven interface.

⸻

🚀 Features
	•	👤 Customer Registration
	•	✈️ Flight Booking System
	•	💺 Seat Selection (Class-based)
	•	🍽️ Meal Preferences
	•	🧳 Extra Baggage Handling
	•	💎 Membership Benefits
	•	🔐 Basic User Verification
	•	🧾 Dynamic Pricing Concept
	•	🖥️ Menu-driven Console Interface

⸻

🛠️ Tech Stack
	•	Language: Java
  
Concepts Used:
	•	Object-Oriented Programming (OOP)
	•	Classes & Objects
	•	Conditional Statements
	•	Loops
	•	Arrays / Data Structures
	•	Modular Code Design

⸻

⸻

⚙️ How to Run
	1.	Clone the repository:

git clone https://github.com/Priyam1312/Airline-Reservation-Management-System-JAVA-.git

	2.	Navigate to the project folder:

cd your-repo-name

	3.	Compile the Java files:

javac *.java

	4.	Run the program:

java Main


⸻

📸 Sample Workflow
	1.	User selects role (Admin / Customer)
	2.	Customer enters details
	3.	Select flight route
	4.	Choose class & seat
	5.	Add extra services
	6.	View final booking summary
